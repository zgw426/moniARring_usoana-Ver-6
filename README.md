## moniARring_usoana-Ver-6 (モニエアリング_ウソ穴 Ver 6.x)

- 凹むARと動画再生を組み合わせたWebARコンテンツ
- 凹むARで、壁に穴が空いたギミックを作る
- 動画と組み合わせると、ユーザーに穴の向こう側が見えるような感覚を提供
